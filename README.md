.env

```
 DATABASE_URL = "postgresql://postgres:password@localhost:5000/ecommerce?schema=ecommerce"
 PORT=PORT
 TOKEN_SECRET=password
```

## Getting started

```
git clone https://github.com/scokljat/e-commerce-server
```

To start the service run:

```
npm install
npm start
```
